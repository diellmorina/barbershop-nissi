# Foto në Cloud - Frizer Nissi

Ky version i website-it është bërë që fotot e portofolios të mos varen nga imazhet lokale.

## Si funksionon

Kur admini zgjedh foto te Admin Paneli:
1. Fotoja bëhet upload në Firebase Storage.
2. Firebase jep një URL online për foton.
3. URL ruhet në Firestore te dokumenti: siteContent/main.
4. Kur dikush hap website-in, fotoja merret nga cloud URL, jo nga folderi lokal.

Pra vizitorët nuk kanë nevojë ta kenë foton të shkarkuar lokalisht.

## Pse jo direkt GitHub?

Mund të bëhet edhe me GitHub, por nuk është ide e mirë nga browser-i sepse duhet GitHub token.
Nëse token vendoset në JavaScript/HTML, çdokush mund ta marrë dhe ta përdorë.
Firebase Storage është më i sigurt për këtë rast.

## Firebase Storage Rules

Firebase Console → Storage → Rules

Vendos:

rules_version = '2';

service firebase.storage {
  match /b/{bucket}/o {
    match /portfolio/{fileName} {
      allow read: if true;
      allow write: if request.auth != null
                   && request.resource.size < 4 * 1024 * 1024
                   && request.resource.contentType.matches('image/.*');
    }
  }
}

Kliko Publish.

## Firestore Rules

Firebase Console → Firestore Database → Rules

Vendos:

rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {
    match /siteContent/{document} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

Kliko Publish.
