# CLOUDINARY SETUP - Frizer Nissi

Ky version i website-it i ruan fotot e portofolios në Cloudinary.

## Si funksionon

Kur admini zgjedh foto te Admin Paneli:
1. Fotoja upload-ohet në Cloudinary.
2. Cloudinary kthen një link online `secure_url`.
3. Linku ruhet në Firestore te dokumenti `siteContent/main`.
4. Kur dikush hyn në website, fotoja shfaqet nga linku online i Cloudinary.

Pra vizitorët nuk kanë nevojë ta kenë foton në folder lokal.

## 1. Krijo unsigned upload preset në Cloudinary

Cloudinary Console → Settings → Upload → Upload presets → Add upload preset

Vendos:
- Signing Mode: Unsigned
- Folder: frizer-nissi/portfolio
- Allowed formats: jpg, jpeg, png, webp
- Max file size: mund ta kufizosh p.sh. 8MB

Ruaje dhe kopjo emrin e upload preset.

## 2. Merr Cloud Name

Cloudinary Console → Dashboard

Kopjo Cloud name.

## 3. Vendos të dhënat në projekt

Hape file-in:

js/cloudinary-config.js

Dhe ndrysho:

cloudName: "YOUR_CLOUD_NAME",
uploadPreset: "YOUR_UNSIGNED_UPLOAD_PRESET",

me vlerat e tua reale.

## 4. Firebase Firestore Rules

Firebase Console → Firestore Database → Rules

Vendos:

rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {
    match /siteContent/{document} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

Kliko Publish.

## 5. Test

1. Hape website-in me Live Server ose hosting.
2. Shtyp /
3. Kyçu si admin.
4. Shko te Fotot - Punimet tona.
5. Zgjidh foto nga PC.
6. Prit mesazhin: Fotoja u upload-ua në Cloudinary dhe u ruajt në cloud.
7. Bëj refresh.
8. Hape në telefon/pajisje tjetër. Fotoja duhet të shfaqet online.
